# FrontEndFrameworkComparison
Front End Framework Comparison: Angular, Vue, React, and Vanilla

## Running React
--From root directory
cd reactjs
npm start
http://localhost:3000

To end session quit terminal with ctrl+c

## Running Angular
--From Root Directory
cd angularjs
ng serve -open
http://localhost:4200

To end session quit terminal with ctrl+c

## Running Vue
--From Root Directory
cd vuejs
npm run serve
http://localhost:8080/ 
To end session quit terminal with ctrl+c
